Overview

Singleton that provides for thread safe date formatting. Out of the box, NSDateFormatters are not thread safe; because of this, you can arrive at a state where a background thread will attempt to use the same formatter at the same time - and this will create what appears to be random crashes. Using this approach, you can cut down the need to instantiate a new date formatter each time you need one by creating on per thread.
